1. D1 : 1936. 1대1 가위바위보

``` py
temp = list(map(int, input().split()))

if temp[0] == 1:
  if temp[1] == 2:
    result = 'B'
  elif temp[1] == 3:
    result = 'A'

if temp[0] == 2:
  if temp[1] == 1:
    result = 'A'
  elif temp[1] == 3:
    result = 'B'

if temp[0] == 3:
  if temp[1] == 1:
    result = 'B'
  elif temp[1] == 2:
    result = 'A'

print(result)
```

2. D1 : 2063. 중간값 찾기

``` py
index = int(input())

temp.sort()

result = temp[(index - 1) // 2]
print(result)
```

4. D2 : 1989. 초심자의 회문 검사

``` py
def reverse(s):
  return s[:] == s[::-1]

num = int(input())
words = []

for i in range(num):
  words.append(input())

for j in range(len(words)):
  if reverse(str(words[j])):
    print(f'#{j+1} 1')
  else:
    print(f'#{j+1} 0')
```
